FRACCIONES

¡Hola! En este nivel seguiremos aprendiendo sobre las fracciones.

PARTE DEL TOTAL

Una fracción puede representar solo una parte del total o varias partes del total.

MULTIPLICAR PARA HALLAR EL TOTAL

Si sabes una parte, multiplica por el denominador.

DIVIDIR PARA HALLAR UNA PARTE

Si sabes varias partes, puedes dividir entre el numerador para saber cuánto vale una.

PARTES DE UNA FRACCIÓN

Recuerda, una fracción tiene dos partes principales: el numerador (arriba) y el denominador (abajo).

FRACCIÓN EQUIVALENTE

Tiene el mismo valor aunque use otros números.

CONSEJO ÚTIL

Cuando una fracción es parte del total, primero divido para saber 1 parte, y luego multiplico para saber todo
